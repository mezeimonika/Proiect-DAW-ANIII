# Proiect-DAW-ANIII

Acest website este despre compania La Roche Posay, o companie de produse skincare.
Website-ul contine 4 pagini: home, produse, abonamente si cosul de cumparaturi.
In pagina de produse am afisat numai produsele care se afla in stoc, in cel de abonamente sunt disponibile 3 tipuri de pachete cu plata recurenta(ori lunara, ori anuala). In cosul de cumparaturi am generat dinamic structura paginii folosind JavaScript, iar la finalul paginii este afisat totalul.
