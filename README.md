# Proiect-DAW-ANIII

Acest website este despre compania La Roche Posay, o companie de produse skincare.
Website-ul contine 4 pagini HTML: home.html, produse.html, abonamente.html si cos.html. JavaScript: script.js, homeScript.js, cart.js. CSS: styles.css.
In pagina de produse am afisat toate produsele disponibile, in cel de abonamente sunt 3 tipuri de pachete cu plata recurenta(ori lunara, ori anuala). 
In cosul de cumparaturi am generat dinamic structura paginii folosind JavaScript, iar la final este afisat pretul total.
